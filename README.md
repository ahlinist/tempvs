## Getting started

To get started you need:
 * Java 1.8
 * Groovy 2.4.11
 * Grails 3.3.2
 * Gradle 4.4.1
 * MySQL 5.7
 * MongoDB 3.2
 
## DB configuration

### MySQL:
Connection parameters are retrieved from env variables:
 * MYSQL_USER
 * MYSQL_PASS
 * MYSQL_HOST
 * MYSQL_PORT
 * MYSQL_DB_NAME

### MongoDB:
Connection parameters are retrieved from env variables:
 * MONGO_USER
 * MONGO_PASS
 * MONGO_HOST
 * MONGO_PORT
 * MONGO_DB_NAME
