## Getting started

To get started you need:
 * Java 1.8
 * Groovy 2.3.6
 * Grails 3.2.11
 * MySQL 5.7
 * MongoDB 3.2
 
## Database configuration

MySQL:
 * User: root (no password)
 * Schema: tempvs

MongoDB:
 * Port: 27017
